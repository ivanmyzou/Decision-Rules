   ```sh
   pip install numpy
   ```

   ```sh
   pip install plotly
   ```

   ```sh
   pip install scipy
   ```     
   ```sh
   pip install ttkthemes
   ```

## Functionalities

### Applying Decision Rules

The program is `Decision_Rules.py`. Decision rules include:
* maxiMax (Value)
* maxiMin (Value)
* miniMax Regret
* Laplace 
* Hurwicz 
* Bayes
